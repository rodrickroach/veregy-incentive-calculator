# Veregy Incentive Calculator

Simple Vite + React web app for estimating incentive payouts.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm install
npm run build
```

## Deploy to Vercel

1. Create a GitHub repository
2. Upload these files
3. Create a free Vercel account with GitHub
4. Import the repository in Vercel
5. Click Deploy

After deployment, open the site in Safari on iPhone and use Share > Add to Home Screen.
